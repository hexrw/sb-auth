How to Use

To use the Vue 3 web component in your project:

1. Install the package

npm install @hexrw/sb-auth

2. Import and register the component

import { registerVueComponent } from '@hexrw/sb-auth'

// Register the Vue 3 component
registerVueComponent()

3. Use in your HTML

<sb-auth-vue></sb-auth-vue>
